# Changelog

All notable changes to SumiJS are documented here.

## 0.1.0

### Added

- FIGlet-inspired text renderer with five built-in fonts.
- Full, fitted, and collision-aware smushed layout modes.
- Immutable shared ASCII grid and render-result API.
- Plain text, ANSI, HTML, SVG, and JSON formatters.
- Gradient styling, alignment, sizing, frames, padding, and margins.
- Image sampling with luminance mapping, color, alpha handling, corrections, thresholding, and dithering.
- Separate browser and optional Sharp-based Node image decoders.
- Zero-runtime-dependency CLI.
- ESM, CommonJS, declarations, subpath exports, tests, CI, package checks, and examples.

# SumiJS documentation

SumiJS converts text and images into reusable ASCII-art grids for websites, terminals, documentation, build pipelines, and command-line workflows. The same rendered result can be formatted as plain text, ANSI, HTML, SVG, or JSON without rendering the source again.

## Start here

- [Getting started](getting-started.md) — install SumiJS and render your first banner.
- [Text rendering](text-rendering.md) — fonts, layout, wrapping, alignment, and multiline input.
- [Image rendering](image-rendering.md) — convert decoded or encoded images into ASCII.
- [Output formats](output-formats.md) — plain text, terminal color, HTML, SVG, and JSON.
- [CLI](cli.md) — generate ASCII output from a terminal or CI job.
- [Interactive Playground](playground.md) — edit browser-safe text and decoded-image examples in Scriptoria.
